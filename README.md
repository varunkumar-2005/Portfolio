# Cloud Portfolio Deployment

This project demonstrates deploying a personal portfolio website using
cloud services as part of the Cloud Computing Laboratory.

The portfolio showcases cloud storage, web hosting, and version control
using real-world tools.

---

##  Technologies Used

- **HTML & CSS** – Frontend portfolio website
- **Supabase** – Object storage for images
- **PythonAnywhere** – Cloud hosting platform
- **Flask** – Backend framework (used earlier for dynamic quote generation)
- **GitHub** – Version control and public repository hosting

---

##  Project Features

- Responsive portfolio website
- Image hosted using **Supabase Storage**
- Public image URL integrated into website
- Deployed using cloud infrastructure concepts
- Public GitHub repository for source code
